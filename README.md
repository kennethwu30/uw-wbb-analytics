# UW Women’s Basketball Analytics Dashboard

A data-driven dashboard for analyzing team and player rebounding performance.  
Built using Python (pandas) for statistical analysis and a web interface (HTML, CSS, JavaScript) for visualization.

---

## Overview

This project transforms raw basketball data into actionable insights by combining a Python data pipeline with a lightweight frontend dashboard.

Using pandas, the system calculates performance metrics such as rebound success rate, player efficiency, and team trends, which are then displayed in an intuitive UI.

---

## Key Features

- **Rebound Performance Metrics**
  - Offensive and defensive rebound percentages
  - Success rates vs total opportunities

- **Top Performer Identification**
  - Automatically highlights the most impactful player

- **Husky Leaders**
  - Players with highest efficiency

- **Development Needed**
  - Players with lower success rates for targeted improvement

- **Full Roster Breakdown**
  - Per-player analysis across different actions:
    - Good execution
    - Stand around
    - On attack

---

## Tech Stack

- **Python (pandas)** — data processing and analysis  
- **HTML / CSS / JavaScript** — frontend dashboard  
- **JSON** — data exchange between backend and UI
